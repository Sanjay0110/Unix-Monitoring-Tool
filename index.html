Backup
Local Backup using rsync

Backup any files in a directory to some other directory locally using rsync. Make sure that the destination directory where the files will backed up should be existing and you have enough disk space (check disk space using "du -h --summarize <directory-name>").

Monitor Processes
Process monitor using top

The top command is used to display all the running and active real-time processes which is updated regularly. It displays CPU usage, Memory usage, Swap Memory, Cache Size, Buffer Size, Process PID, User, Commands, high memory and cpu utilization of running processes. You can terminate the processes using "kill -9 <Process-Id>".
